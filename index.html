MY EXPERTISE

HTML

HTML, or HyperText Markup Language, is a standard markup language used to create the structure and content of web pages on the internet.

CSS

CSS, or Cascading Style Sheets,is a style sheet language used for describing the presentation and visual formatting of HTML documents. It is a fundamental technology in web development that allows designers and developers to control the appearance and layout of web pages.

FIGMA

Figma is a collaborative design and
prototyping tool that is primarily used in the field of user interface (UI) and user experience (UX) design. It enables designers and design teams to create, share, and iterate on digital designs for websites, web applications, mobile apps, and other digital products.

BOOTSTRAP

Bootstrap is a popular open-source front-end framework for web development. It was originally created by Twitter and is now maintained by the Bootstrap community. Bootstrap is designed to make it easier and faster to create responsive and visually appealing websites and web applications.

JAVA SCRIPT

JavaScript is often referred to as a scripting language because it is typically executed within a web browser. However, it can also be used on the server-side (e.g., with Node.js) to build full-stack applications.
